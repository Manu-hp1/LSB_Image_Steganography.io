Secure Data Concealment Inside Images

A low-level C implementation of Least Significant Bit steganography engineered for secure message embedding within 24-bit BMP images.

Watch Demo

Project Overview

StegaSecure enables confidential data embedding inside bitmap images using precise bit-level manipulation. The implementation ensures visual integrity while securely storing hidden information.

Core Capabilities

Bit-Level Precision

Manipulates only the least significant bits of pixel bytes.

Modular C Architecture

Separate encoding, decoding, and validation modules.

Image Integrity Preservation

No visible distortion after data embedding.

Core Encoding Logic

pixel_byte = pixel_byte & 0xFE;   // Clear LSB
pixel_byte = pixel_byte | bit;    // Insert secret bit
