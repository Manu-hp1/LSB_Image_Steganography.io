Secure Data Concealment Inside Images

A low-level C implementation of Least Significant Bit steganography engineered for secure message embedding within 24-bit BMP images.

Watch Demo

Product Overview

StegaSecure embeds confidential data inside bitmap images using precise bit-level manipulation while preserving image quality and structural integrity.

Problem Statement

Sensitive communication over digital channels may expose data to interception. This project solves that problem by embedding confidential messages directly into image files at binary level.

System Workflow

1. Validation

Verify BMP header and confirm 24-bit image structure.

2. Encoding

Clear LSB → Insert secret bit → Write modified byte.

3. Decoding

Extract LSB → Reconstruct message → Output secret text.

Core Encoding Logic

pixel_byte = pixel_byte & 0xFE;   // Clear LSB
pixel_byte = pixel_byte | bit;    // Insert secret bit
