LSB Image Steganography

A secure data concealment system built in C that embeds confidential messages inside 24-bit BMP images using precise bit-level manipulation.

Source Code Watch Demo

Why StegaSecure?

Binary-Level Security

Modifies only the least significant bits of pixel data for invisible embedding.

Image Integrity Preserved

No visible distortion while embedding hidden information.

Modular C Architecture

Separate encoding, decoding, and validation modules for clarity.

Key Features

  • 24-bit BMP format validation
  • Bitwise masking using AND & OR operators
  • Efficient file handling with fread and fwrite
  • Command-line execution
  • Structured modular code design

Technology Stack

Linux C Programming Bitwise Operations File Handling Pointers Structures
