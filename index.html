LSB Image Steganography

Secure Data Hiding Using Bit-Level Manipulation in C

Project Overview

Implemented Least Significant Bit (LSB) steganography in C to embed secret text data within 24-bit BMP images. The technique modifies only the least significant bits of pixel bytes, ensuring zero visible distortion while maintaining image integrity.

Technical Implementation

  • 24-bit BMP format validation
  • Binary-level pixel manipulation
  • Bit masking using bitwise operators
  • Separate encoding and decoding modules
  • Efficient file handling (fread, fwrite)
  • Command-line interface execution
